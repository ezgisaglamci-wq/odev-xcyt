# CYPEX — Denizcilik Lojistik Zekası

Modern gemi brokerlığı için tasarlanmış yapay zeka destekli komuta merkezi. Armatörleri ve kiracıları akıllı eşleştirme, sefer takibi, müzakere odaları ve piyasa analizi araçlarıyla buluşturur.

## Özellikler

- Yapay zeka destekli gemi-yük eşleştirme
- Canlı pazar yeri ve açık kapasite görünümü
- Filo yönetimi ve sertifika takibi
- Sefer takibi ve demurraj projeksiyonu
- Müzakere odası ve dijital teklif altyapısı

## Yerelde Çalıştırma

**Gereksinimler:** Node.js 20+

```bash
pnpm install
pnpm dev
```

Tarayıcıda `http://localhost:3000` adresini açın.

## Üretim Build

```bash
pnpm build
pnpm preview
```
